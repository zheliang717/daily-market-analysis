## 二、政策与全球事件量化影响分析

### ⚠️ 事件1：国际油价突破100美元 → 全球通胀压力回升
- **影响路径：** 地缘紧张+OPEC+减产 → 布伦特突破100美元 → 全球能源成本上升 → 通胀预期升温 → 央行降息空间收窄
- **量化估计：** 油价每上涨10美元/桶，全球CPI约推升0.3-0.5个百分点。若布伦特维持100美元以上，美国CPI可能回升至4%+，降息预期大幅推迟。A股能源板块盈利弹性+15%-25%，但航空/化工成本端承压3%-8%。
- **核心风险：** 若中东局势进一步升级或OPEC+意外扩大减产，油价可能冲击110-120美元，触发新一轮"滞胀"交易。
- **数据来源：** 中国基金报、ICE、NYMEX

### ✅ 事件2：CPO技术路线加速 → AI算力产业链新一轮爆发
- **影响路径：** AI大模型持续迭代 → 算力需求指数级增长 → 传统光模块功耗/带宽瓶颈 → CPO技术加速落地 → 光通信产业链全面受益
- **量化估计：** CPO市场规模预计2026-2028年CAGR超50%。相关标的短期弹性10%-30%。光模块龙头订单能见度已延伸至2027年。
- **核心风险：** 技术路线竞争（LPO等替代方案）、量产良率不及预期、估值过高后回调风险。
- **数据来源：** 央视财经、腾讯新闻

### ℹ️ 事件3：特斯拉Q1财报 → AI叙事关键验证
- **影响路径：** 财报公布 → 若AI/Robotaxi超预期 → 科技股情绪提振；若不及预期 → 新能源+AI板块承压
- **量化估计：** 预计营收约223亿美元。若Robotaxi商业化时间线提前，股价可能反弹5%-10%；若交付量/利润率下滑超预期，可能拖累板块3%-5%。
- **核心风险：** 马斯克政府角色分散精力、关税政策不确定性、竞争加剧。
- **数据来源：** 新浪财经、富途牛牛

- **影响路径：** 全球央行持续购金+美元信用下降+地缘风险 → 黄金持续创新高 → 各国储备多元化加速
