## 二、全球局势分析

### 2.1 中东冲突显著升级（核心变量）

**最新进展（3/22）：**
- 美国和以色列对伊朗军事打击进入第22天
- 伊朗宣布发动"真实承诺-4"行动第72波攻势，声称成功打击以色列北部、核心地带及美第五舰队目标
- 伊朗军方正在霍尔木兹海峡采取重大行动
- **五角大楼向中东增兵超两千人，制定向伊朗出动地面部队详细计划**
- 特朗普发表强硬言论

来源：新京报, 2026-03-22 / 凤凰网, 2026-03-22 / 每日经济新闻, 2026-03-22

**市场影响：**
- 自2/28冲突升级以来，标普500累计跌5.4%，纳指跌4.5%，蒸发5.5万亿美元（来源：新浪财经/网易, 3/22）
- 油价受地缘风险溢价支撑，布伦特维持$104-110区间
- 避险情绪推动金价一度冲高至$4,648后回落

### 2.2 美联储3月FOMC：维持利率不变

3月19日凌晨，美联储以11:1投票维持联邦基金利率目标区间3.50%–3.75%不变，延续每月400亿美元国债购买。对关税通胀风险措辞更趋谨慎，降息路径不确定性上升。

来源：中信证券研究 / 中信证券固定收益部 / 东海证券, 2026-03-19

### 2.3 全球贸易：美国新15%关税

美国财长贝森特称新的15%关税税率将很快实施，全球贸易进入新一轮博弈。美股连跌四周，关税不确定性为重要压制因素。

来源：中国经济网, 2026-03-21 / 彭博社

### 2.4 欧洲央行：加息预期升温

巴克莱3月20日上调预期，预计ECB在4月和6月各加息25个基点（此前预测维持不变）。欧美货币政策分化加剧，欧洲股市本周大幅下挫。

来源：财联社/凤凰网, 2026-03-20

- 但地缘风险溢价主导短期油价走势

## 三、A股板块分析

> 本周A股：沪指周跌3.38%（近几个月第二大周跌幅），破位4000点整数关口；平均股价周跌4.3%；创业板指周涨1.26%，光伏/储能板块逆势走强。（来源：网易/今日头条, 2026-03-21~22）

### 科技板块

**本周表现**：CPO概念爆发，半导体走强；AI应用、算力租赁走弱。美股科技承压（纳指周跌2.07%）。

**驱动**：CPO技术突破；美联储高利率压制估值；AI应用端炒作降温。

**产业链**：
- 上游（芯片/GPU）：需求旺盛但估值高位
- 中游（光模块/CPO）：算力基建景气度最高
- 下游（AI应用）：商业化不及预期

**价值投资潜力股**：
1. **中际旭创（300308）** — 全球光模块龙头，CPO技术领先，海外订单饱满
2. **海光信息（688041）** — 国产CPU/GPU核心标的，信创+AI双轮驱动
3. **北方华创（002371）** — 半导体设备龙头，国产替代+先进制程突破

### 新能源板块

**本周表现**：光伏设备板块涨幅居前，储能走强，锂电池走强。创业板指周涨1.26%主要由新能源贡献。（来源：证券时报/凤凰网, 3/20）

**驱动**：光伏设备出口超预期；政策持续推进装机目标；锂电产业链价格企稳。

**产业链**：
- 上游（硅料）：价格下行至底部，利润向中下游转移
- 中游（电池片/组件）：盈利能力改善，TOPCon/HJT技术迭代加速
- 下游（装机/储能）：装机量高增，储能需求爆发

**价值投资潜力股**：
1. **隆基绿能（601012）** — 全球硅片+组件龙头，BC电池技术储备深厚
2. **宁德时代（300750）** — 全球动力电池绝对龙头，储能业务高速成长
3. **阳光电源（300274）** — 光伏逆变器+储能系统双龙头，海外收入占比高

### 消费板块

**驱动**：两会聚焦扩内需；AI终端创新驱动消费电子。

## 六、风险提示

**宏观风险**
1. 中东局势进一步升级：美军增兵2000+并制定地面部队计划，若冲突扩大，油价可能突破$120
2. 霍尔木兹海峡：伊朗在海峡采取重大行动，全球能源运输通道面临威胁
3. 美联储政策：若通胀因油价反弹，可能重新收紧，全球流动性承压
4. 全球贸易战：新关税+反制措施，拖累全球经济增长

**行业风险**
1. 科技股估值：AI概念部分标的估值过高，业绩不及预期可能回调
2. 新能源产能过剩：光伏、锂电池产能扩张过快，价格战持续
3. 有色金属波动：LME铜单日暴跌$249，地缘+美元+需求不确定性叠加

**下周关注**
1. 3月23日（周一）A股开盘：周末中东局势升级，下周一开盘承压
2. 3月23日24时：国内成品油调价窗口
3. 美股是否出现超跌反弹

**投资建议**
1. 防御为主：配置高股息资产
2. 精选成长：聚焦核心技术壁垒和稳定现金流龙头
3. 分散配置：跨行业、跨市场分散，降低单一风险
4. 关注政策：美联储后续议息、两会政策落地、中美贸易谈判、中东局势演变
